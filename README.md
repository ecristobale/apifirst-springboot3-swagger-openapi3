## DESCRIPTION
The **goal** of this project is to create a simple app developed with **API First** and by using **Swagger**, **Spring Boot 3** and **Java 17** . 
* Exposed endpoint: `/greetings`
* Swagger UI: `/swagger-ui/index.html`
